# Wudu Robot 🤖

An interactive drag-and-drop programming game that teaches children the core idea of programming: **a computer does exactly what you tell it — no more, no less.**

Built for the *Tech Titans* summer camp at Al Hikmah International Private Training Institute, for boys aged 9–11.

## What it teaches

Children arrange 8 instruction blocks into the correct order to program a robot to perform wudu. When they press **RUN**, the robot performs *exactly* the sequence they built:

- **Wrong order?** The robot follows the instructions faithfully, then gets gently confused and stops — showing that the mistake was in the instructions, not the machine.
- **Correct order?** The robot completes wudu, celebrates, and reveals a "behind the curtain" panel showing how their blocks became real-looking code.

This mirrors the single most important lesson in early programming: computers can't guess your intent — they only follow your steps.

## The 8 steps

1. Intention (Niyyah)
2. Bismillah
3. Wash Hands
4. Mouth and Nose
5. Wash Face
6. Wash Arms
7. Wipe Head and Ears (once)
8. Wash Feet

## How it's built

A single self-contained `index.html` file — HTML, CSS, and JavaScript, no dependencies, no build step. Works offline and on any modern browser, including tablets and phones.

## Run it locally

Just open `index.html` in any web browser. That's it.
